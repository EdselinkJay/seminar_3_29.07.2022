## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочкой (*) или знаком (+)
Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3
+ Элемент 4

Чтобы добавить нумерованные списки, необходмо пункты просто пронумеровать. Например вот так:
1. Первый пункт
2. Второй пункт
